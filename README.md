# Gemini AI Smart Browser

This is a web application that acts as a "Smart Browser." It uses the **Google Gemini 2.5 Flash API** to provide detailed, AI-generated insights for any search topic.

## Features

-   **AI-Powered Explanations:** Get deep dives into topics, including definitions, history, and key facts.
-   **Clean UI:** A "browser-in-browser" interface built with **Bootstrap 5**.
-   **Real-time Feedback:** Includes loading spinners and error handling.
-   **Responsive Design:** Works on desktops, tablets, and mobile phones.

## Tech Stack

-   **Frontend:** HTML5, CSS3, JavaScript (ES6+)
-   **Styling:** Bootstrap 5 (CDN)
-   **API:** Google Generative AI (Gemini 2.5 Flash)

## How to Run

1.  Clone this repository:

    ```bash
    git clone https://github.com
    ```
2.  Open `index.html` in any modern web browser.
3.  Enter a topic in the search bar and click **Search**.

## Security Note

The current version has the API Key hardcoded for demonstration. For production use, it is recommended to use environment variables or a backend proxy to keep your **API\_KEY** secure.
